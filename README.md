# AI Job Market Analytics Dashboard 2026

## Overview
This project visualizes the AI job market landscape for 2026, focusing on salary benchmarks, industry trends, and hiring velocity.

## Key Insights
- Remote work captures 26% of the total AI market share.
- Average AI package is ₹24.82 LPA, outpacing traditional IT roles.
- Hiring Velocity: 59% of AI roles are currently open and actively hiring.

## Tech Stack
- Microsoft Excel (Data Cleaning, Pivot Tables, Visualization)

Download the `AI_Job_Market_Analytics_2024_2025.xlsx` file to explore the interactive dashboard and raw data analysis.
